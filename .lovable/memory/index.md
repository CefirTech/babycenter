# Project Memory

## Core
E-commerce vêtements enfants 0-16 ans, boutique "BABYCENTER", français uniquement.
Design premium féminin chaleureux: Playfair Display headings, DM Sans body. Rose/gold/cream palette.
Primary hsl(350 45% 55%), accent gold hsl(38 70% 55%), bg cream hsl(30 33% 98%).
Devise: FCFA. Localisation: Côte d'Ivoire, Abidjan. WhatsApp: canal de vente clé.
Paiements cibles: Orange Money, Moov Money, MTN Money, Wave, carte bancaire (CinetPay/Flutterwave).
